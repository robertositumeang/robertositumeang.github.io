Tools

Excel/Spreadsheet

A spreadsheet tool used for organizing, analyzing, and visualizing data. Easy to use for small datasets.

Python

A programming tool used to clean, analyze, and visualize data. It helps automate tasks and handle big datasets.

SQL

A language used to get and manage data from databases. Analysts use it to find insights and combine data from different tables.

Tableau

A tool for creating charts and dashboards to make data easy to understand. Great for sharing insights.

Google Colab

An online tool to write and run Python code. Good for working with data and sharing work with others.

Power BI

Microsoft’s tool for making reports and dashboards. Works well with Excel and other Microsoft products.

PowerPoint / Google Slides

Tools for creating presentations. Analysts use them to share insights, reports, and data stories with teams and stakeholders.

Google Data Studio (Looker Studio)

A free tool for building interactive dashboards. It connects to multiple data sources and helps visualize key metrics in real time.

Google Analytics

A tool for tracking website traffic and user behavior. It helps analysts understand visitors, measure performance, and improve online marketing strategies.
